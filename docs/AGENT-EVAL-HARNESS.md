# Agent Eval Harness

## Purpose

Measure whether agents are accurate, useful, and aligned with the platform rules.

## What to Score

- factual accuracy
- code correctness
- compliance awareness
- refusal quality
- drift from project context
- clarity of next step

## Rules

- use repeatable prompts
- compare outputs against known-good cases
- keep a regression set for important workflows
- do not ship a new agent mode without scoring it first
