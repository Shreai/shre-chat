# Launch Approval Checklist

## Required Signoff

- product owner
- engineering
- QA
- security
- compliance if applicable
- legal review if the product crosses jurisdictions, uses custom terms, or handles regulated data
- support/docs
- trace-route review for visible changes

## Checklist

- scope matches the request
- tests passed
- build passed
- evidence attached
- rollback plan documented
- compliance register filled
- jurisdiction matrix filled when the product is sold or deployed outside a single known legal environment
- terms, privacy, and IP docs reviewed for the launch market
- handoff pack ready
- support contact known
