# Data Classification

## Levels

- **Public**: safe to expose publicly
- **Internal**: team-only operational data
- **Confidential**: customer, product, or business-sensitive data
- **Regulated**: PHI, payment data, or any legally controlled data

## Rules

- classify new data before storing it
- keep regulated data out of client bundles
- separate public docs from confidential operational records
- define retention and deletion by classification
