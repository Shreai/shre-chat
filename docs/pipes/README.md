# Pipes

Starter docs for app-to-app pipes in Shre OS.

## Use

Copy the pipe template for each integration and fill in the source, destination, payload, retries, and rollback path.

## Files

- [Pipe Manifest](../PIPE-MANIFEST.md)
- [App README Template](../APP-README-TEMPLATE.md)

## Rule

Any pipe that moves data between apps must have a manifest row before it ships.
