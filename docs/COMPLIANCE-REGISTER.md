# Compliance Register

## Purpose

Track which compliance profile applies to each product and workspace.

## Profiles

- None / internal only
- HIPAA
- PCI DSS
- SOC 2
- ISO/IEC 27001
- Conexxus-aligned retail/fuel product
- Custom customer policy

## Rules

- record the profile before build starts
- record the owner for the profile
- record the evidence needed for launch
- do not claim certification unless it is formally obtained

## Fields

- product name
- owner
- environment
- profile
- in-scope data types
- evidence links
- approval status
